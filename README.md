# Место

Данный проект включает в себя лендинг, контент которого помогает узнать красивые и интересные места России.

В проекте используются следующие технологии:

* HTML
* CSS (Grid, Flex)
* БЭМ (Nested)
* Адаптивная верстка
* JavaScript
